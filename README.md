# 🎵 Songs – Music Player Web App

A clean, neon-themed music player built as a side project. Hosted on GitHub Pages.

🔗 **Live demo:** [https://aadi-playz23.github.io/Songs/](https://aadi-playz23.github.io/Songs/)

## Features
- Play/pause, next/previous
- Search by song title or artist
- Genre filtering
- Responsive layout

## Tech stack
- HTML5 / CSS3 / JavaScript (vanilla)
- Hosted on GitHub Pages

## How to run locally
```bash
git clone https://github.com/aadi-playz23/Songs.git
cd Songs
# Open index.html in your browser
